The Process

How We Scout

Every report is built from raw box score data — at-bats, hits, strikeouts, walks, earned runs — compiled game by game across the full season.

Player cards include scouting grades (S through D) across five tools: Contact, Power, Plate Discipline, Speed, and Overall.

What's Tracked

Stat Glossary

  • PA — Plate Appearances (AB + BB + HBP + SF)
  • AVG — Batting Average (H ÷ AB)
  • OBP — On-Base Percentage
  • SLG — Slugging Percentage (TB ÷ AB)
  • OPS — On-Base + Slugging
  • ERA — Earned Run Average (ER ÷ IP × 9)
  • WHIP — Walks + Hits per Inning Pitched
  • SB — Stolen Bases
